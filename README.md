# D&D DM Inspiration Toolkit

A single-file D&D Dungeon Master companion app — session management, party tracker, generators, and more.

## Live App
👉 **[Open the Toolkit](https://YOUR-USERNAME.github.io/dndit)**

## Deploy Your Own

1. Fork or clone this repo
2. Go to **Settings → Pages**
3. Source: **Deploy from a branch**
4. Branch: **main**, folder: **/ (root)**
5. Click Save — live in ~1 minute

## Data & Privacy
All data is stored locally in your browser via `localStorage`. Nothing is sent to any server.
